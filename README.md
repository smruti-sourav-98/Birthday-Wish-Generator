🎂 Birthday Wish Generator
A simple and fun web application that generates personalized birthday wishes with a single click! Spread joy and make someone's special day even more memorable with creative, heartfelt, and customizable birthday messages.

✨ Features
🎈 Random Wish Generator – Get a unique birthday wish every time

📝 Personalized Messages – Add the recipient's name for a custom touch

🎨 Beautiful UI – Clean, modern, and responsive design

📋 Copy to Clipboard – Easily copy wishes to share anywhere

🔄 Regenerate – Not happy with the wish? Generate a new one instantly

📱 Mobile Friendly – Works perfectly on all devices

🌈 Fun & Engaging – Includes emojis and cheerful animations

🧠 How It Works
The generator uses a curated list of birthday messages with placeholders for names. When a user clicks "Generate," the app:

Takes the input name (or uses a default if empty)

Selects a random wish from the database

Replaces {name} with the recipient's name

Displays the wish with a smooth animation

Enables copy functionality for easy sharing
